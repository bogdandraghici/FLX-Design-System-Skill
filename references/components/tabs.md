Tabs organizes related content into panels the user can switch between without leaving the page. It accepts the following props: size ("small" or "medium", defaulting to "medium"), inverted (boolean, default false), tabs (string array), activeIndex (number, default 1), hasIcon (boolean, default false), and hasCounter (boolean, default false). Tabs has two states: active and inactive. It comes in two sizes: small and medium. Its anatomy consists of a TabsContainer laid out horizontally with a full-width bottom divider, TabCells with no top padding, 12px left/right padding, bottom padding of 8px at medium and 6px at small, and 6px gap between children, an optional TabIcon at 16x16, a TabName text label, and an optional Counter pill badge with 99px corner radius. At medium size the tab name uses 14px/24px font. At small size the tab name uses 12px/16px font. Typography uses Open Sans. The active tab name is weight 600 (SemiBold) at both sizes. The inactive tab name at medium is weight 600, and at small is weight 400 (Regular). In the light theme, the container divider is #e3e8ed at 1px. The active tab has a 2px bottom stroke in #006bd8 with #006bd8 text. Inactive tabs have no stroke with #64748b text. The active counter badge has a #006bd8 fill with #ffffff text. The inactive counter badge has a #e3e8ed fill with #64748b text. In the dark/inverted theme, the container divider is #475263 at 1px. The active tab has a 2px bottom stroke in #006bd8 with #006bd8 text. Inactive tabs have #64748b text. The active counter badge has a #006bd8 fill with #ffffff text. The inactive counter badge has a #e3e8ed fill with #64748b text.
