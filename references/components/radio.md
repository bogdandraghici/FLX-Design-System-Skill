Radio is a selection control that lets users pick exactly one option from a mutually exclusive set. It accepts the following props: selected (boolean, default false), state ("default", "error", or "disabled", default "default"), border (boolean, default true), inverted (boolean, default false), size ("small" or "medium", default "medium"), label (string), value (string), and subtitle (string, default ""). The radio has three states: default, error, and disabled. It comes in two sizes: small and medium. Its anatomy consists of a Label, an Input Container with 8px radius, a Radio Icon (circle), Value text, and an Error Icon. At medium size the container is 36px tall with 8px left padding and 12px right padding, 8px gap, and a 24x24 radio icon. At small size the container is 28px tall with a 16x16 radio icon. Typography matches the checkbox: at medium size the label is 14px/24px weight 600 and the value is 14px/24px weight 400, at small size the label is 12px/16px weight 600 and the value is 12px/16px weight 400. When a subtitle is present the value uses weight 600 and the subtitle uses 12px/16px weight 400. In the light theme when unselected in default state, the container background is #ffffff with #e3e8ed border, radio stroke is #cbd1db, and value text is #1d232c. When selected, the container background becomes #e6f0fb with #006bd8 border, radio fill is #006bd8, and value remains #1d232c. On hover in the default state, the unselected border becomes #cbd1db and radio stroke #a6b0be, and the selected border and radio fill become #005fc0. In error state when unselected, the container background is #ffffff with #eb4e33 border. In error state when selected, the container background is #fde9e6 with #e62200 border and radio fill #e62200. On hover in the error state, the border and radio fill become #d11f00. When disabled, the container background is #e3e8ed for both selections, with border #e3e8ed when unselected and #64748b when selected, radio stroke #64748b when unselected, radio fill #64748b when selected, and value text #64748b. In the dark theme when unselected in default state, the border is #64748b, radio stroke is #64748b, and value text is #ffffff. When selected, the container background is #002d5b with #3389e0 border and radio fill #3389e0. On hover in the dark default state, the unselected border and radio stroke become #8390a2, and the selected border and radio fill become #549ce5. In the dark theme in error state, the border is #eb4e33 for both selections (#ee6b54 on hover); when unselected the container background is transparent, and when selected it is #610e00 with radio fill #e62200 (#d11f00 on hover). In the dark theme when disabled, the container background is #475263 with #5b6a7e border for both selections, radio stroke is #64748b when unselected, radio fill (dot) is #a6b0be when selected, and value text is #8390a2. In the error state the radio circle stroke and the value text keep their default colors; only the container border and fill and the selected radio fill turn red. Note that the radio's unselected error border is red-400 (#eb4e33) while the checkbox uses red-500 (#e62200) for the same state; this difference is intentional.
