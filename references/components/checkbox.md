Checkbox is a selection control that lets users select one or more options from a list, or toggle a single setting on/off. It accepts the following props: selected (boolean, default false), indeterminate (boolean, default false), state ("default", "error", or "disabled", default "default"), border (boolean, default true), inverted (boolean, default false), size ("small" or "medium", default "medium"), label (string), value (string), and subtitle (string, default ""). The checkbox has three states: default, error, and disabled. It also supports an indeterminate selection: set indeterminate on a parent checkbox when only some of its children are selected. Indeterminate shares the filled look with selected (same fills, borders, and hover colors); only the icon differs, showing a white horizontal dash instead of a check. Clicking it selects all children; clicking again deselects them. It comes in two sizes: small and medium. Its anatomy consists of a Label, an Info Icon, an Input Container with 8px radius, a Checkbox Icon (square with rounded corners), Value text, and an Error Icon. At medium size the container is 36px tall with 8px horizontal / 6px center padding, a 24x24 checkbox icon with 4.8px corner radius. At small size the container is 28px tall with 6px horizontal / 4px center padding, a 16x16 checkbox icon with 3.2px corner radius. Typography at medium size uses 14px/24px weight 600 for the label and 14px/24px weight 400 for the value. At small size it uses 12px/16px weight 600 for the label and 12px/16px weight 400 for the value. When a subtitle is present the value uses weight 600 and the subtitle uses 12px/16px weight 400. In the light theme when unselected in default state, the container background is #ffffff with #e3e8ed border, checkbox stroke is #cbd1db, and value text is #1d232c. When selected, the container background becomes #e6f0fb with #006bd8 border, checkbox fill is #006bd8, and value remains #1d232c. On hover in the default state, the unselected border becomes #cbd1db and checkbox stroke #a6b0be, and the selected border and checkbox fill become #005fc0. In error state when unselected, the container background is #ffffff with #e62200 border and value is #1d232c. In error state when selected, the container background is #fde9e6 with #e62200 border and checkbox fill is #e62200. On hover in the error state, the border and checkbox fill become #d11f00. When disabled and unselected, the container background is #e3e8ed with rgba(15,17,20,0.1) border, checkbox stroke is rgba(15,17,20,0.1), and value is #64748b. When disabled and selected, the container background is #e3e8ed with #64748b border, checkbox fill is #64748b, and value is #64748b. In the dark/inverted theme when unselected in default state, the container background is transparent with #64748b border, checkbox stroke is #64748b, and value text is #ffffff. When selected, the container background is #002d5b with #3389e0 border, checkbox fill is #3389e0, and value is #ffffff. On hover in the dark default state, the unselected border and checkbox stroke become #8390a2, and the selected border and checkbox fill become #549ce5. In error state when unselected, the container background is transparent with #eb4e33 border. In error state when selected, the container background is #610e00 with #eb4e33 border and checkbox fill is #eb4e33. On hover in the dark error state, the border and checkbox fill become #ee6b54. The error icon stroke in the dark theme is #eb4e33 (it is #e62200 in the light theme). When disabled and unselected, the container background is #475263 with #5b6a7e border and value is #8390a2. When disabled and selected, the container background is #475263 with #8390a2 border, checkbox fill is #a6b0be, and value is #8390a2.
