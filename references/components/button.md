Button is a clickable component that triggers actions like submitting forms, confirming dialogs, or navigating between steps. It accepts the following props: scope ("Brand", "Danger", or "Success", defaulting to "Brand"), variant ("Primary", "Secondary", or "Tertiary", defaulting to "Primary"), state ("Default", "Hover", "Pressed", or "Disabled", defaulting to "Default"), size ("Medium", "Small", "XS", or "XXS", defaulting to "Medium"), inverted (boolean, default false), hasLabel (boolean, default true), hasIconStart (boolean, default false), hasIconEnd (boolean, default false), and label (string, default "Button"). The button has four states: Default, Hover, Pressed, and Disabled. It comes in four sizes: XXS, XS, Small, and Medium. Its anatomy consists of a Container laid out horizontally with center alignment, an optional Leading Icon, a Label, and an optional Trailing Icon. At Medium size the button is 36px tall with 14px/24px font, 8px corner radius, 6px item spacing, 24px icon size, and 12px horizontal / 6px vertical padding. At Small size it is 28px tall with 12px/16px font, 8px corner radius, 6px item spacing, 16px icon size, and 8px horizontal / 6px vertical padding. At XS size it is 24px tall with 10px/12px font, 8px corner radius, 6px item spacing, 12px icon size, and 6px horizontal / 6px vertical padding. At XXS size it is 16px tall with 10px/12px font, 4px corner radius, 2px item spacing, 12px icon size, and 4px horizontal / 2px vertical padding.
