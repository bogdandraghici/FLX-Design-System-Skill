Input Field is a single-line text input for collecting free-form data like names, emails, or numbers. It accepts the following props: state ("default", "focused", "error", or "disabled", default "default"), filled (boolean, default false), size ("small" or "medium", default "medium"), inverted (boolean, default false), hasTopLabel (boolean, default true), hasSideLabel (boolean), hasIconStart (boolean), hasIconEnd (boolean), hasPrefix (boolean), hasSuffix (boolean), hasDescription (boolean), placeholder (string), and value (string). The input field has five states: default, hover, focused, error, and disabled. It comes in two sizes: small and medium. Its anatomy consists of a Label, an Info Icon, an Input Container with 8px radius, Icon Start, Prefix, Input Text or Placeholder, Suffix, Icon End, and a Description. At medium size the container is 36px tall with 0px top/bottom and 8px left/right padding, 6px gap, and 8px corner radius. At small size the container is 28px tall. Typography at medium size uses 14px/24px weight 600 for the label and 14px/24px weight 400 for the value and placeholder. At small size it uses 12px/16px weight 600 for the label and 12px/16px weight 400 for the value and placeholder. The description uses 12px/16px weight 400 at all sizes. In the light theme the default state has a #ffffff container fill with #e3e8ed stroke, #1d232c value text, #64748b placeholder, and #6b7789 description. On hover the stroke changes to #cbd1db. When focused the stroke becomes #006bd8. In error state the stroke is #e62200 and the description turns #e62200. When disabled the fill becomes #e3e8ed with #e3e8ed stroke, value is #5b6a7e, placeholder is #8390a2, label is #8390a2, and description is #64748b. In the dark theme the default state has a transparent container fill with #64748b stroke, #ffffff value text, #a6b0be placeholder, and #8390a2 description. On hover the stroke changes to #8390a2. When focused the stroke becomes #006bd8. In error state the stroke is #eb4e33 and the description turns #e62200. When disabled the fill is #475263 with #5b6a7e stroke, value is #cbd1db, and label is #64748b. Mark required fields with an asterisk after the label so users can scan a form and see what they must fill in before submitting; use neutrals-500 for the asterisk so it reads as a quiet annotation rather than an alert.
