Select Field is a dropdown for choosing one or more options from a predefined list, with chip-based multi-select support. It accepts the following props: state ("default", "focused", "error", or "disabled", default "default"), filled (boolean, default false), fillMode ("text" or "chips", default "text"), size ("small" or "medium", default "medium"), inverted (boolean, default false), hasLabel (boolean, default true), hasDescription (boolean), hasPrefix (boolean), hasSuffix (boolean), hasIconStart (boolean), and placeholder (string). The select field has four states: default, focused, error, and disabled. It comes in two sizes: small and medium. Its anatomy consists of a Label, an Info Icon, an Input Container with 8px radius, Icon Start, Prefix, Placeholder or Chips, Suffix, Error Icon, Caret Down, a Chip Group, individual Chips, a Chip Count, and a Description. At medium size the container is 36px tall with 6px top/bottom and 12px left/right padding, 6px gap, 8px corner radius, a 24x24 caret, and chips with 8px corner radius and 2px vertical, 6px horizontal padding. At small size the container is 28px tall with a 16x16 caret and chips with 6px corner radius. When filled, single-select fields use text mode and render the selected value as plain text; multi-select fields use chip mode so users can see and remove individual selections. Typography at medium size uses 14px/24px weight 600 for the label, 14px/24px weight 400 for the value and placeholder, and 14px/24px for chip text. At small size it uses 12px/16px for the label and 12px/16px for value, placeholder, and chip text. The description uses 12px/16px at all sizes. In the light theme the default state has a #ffffff container fill with #e3e8ed stroke, #1d232c value text, #64748b placeholder, #e3e8ed chip background, #000000 chip text, #006bd8 chip count, and #6b7789 description. On hover the stroke changes to #cbd1db. When focused the stroke becomes #006bd8. In error state the stroke is #e62200 and description turns #e62200. When disabled the fill is #e3e8ed with #e3e8ed stroke, value is #5b6a7e, placeholder is #8390a2, chip background is #cbd1db, and chip text is #475263. In the dark theme the default state has no fill (transparent) with #64748b stroke, #ffffff value text, #a6b0be placeholder, #475263 chip background, #ffffff chip text, and #8390a2 description. On hover the stroke changes to #8390a2. When focused the stroke becomes #006bd8. In error state the stroke is #eb4e33 and the description turns #e62200. When disabled the fill is #475263 with #5b6a7e stroke, value is #cbd1db, chip background is #5b6a7e, and chip text is #cbd1db.
