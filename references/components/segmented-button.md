Segmented Button switches between 2-5 mutually exclusive views or filters. It accepts the following props: size ("small" or "medium", defaulting to "medium"), inverted (boolean, default false), disabled (boolean, default false), options (string array), selectedIndex (number, default 0), label (string), and hasLabel (boolean, default true). The segmented button has two states: default and disabled. It comes in two sizes: small and medium. Its anatomy consists of a Root laid out vertically with 2px gap, a Label above the container, a Container (pill-shaped track holding all segments), and individual Segments (selectable buttons inside the container). At medium size the container is 36px tall with 4px padding, 4px gap between segments, and 8px corner radius. Each medium segment is 73x28 with 6px corner radius. At small size the container is 28px tall with 4px padding and 2px gap between segments. Each small segment is 65x20 with 6px corner radius. Typography uses Open Sans. The label is weight 600 (SemiBold) at 14px/22px for medium and 12px/16px for small. Segment text is weight 400 (Regular) at 14px/22px for medium and 12px/16px for small. In the light theme, the container fill is #e3e8ed. The selected segment has a #006bd8 fill with #ffffff text. Unselected segments have transparent fill with #475263 text. Unselected segments on hover use rgba(255,255,255,0.6) fill. When disabled, the selected segment fill is #8390a2 with #ffffff text. The label color is #1d232c. In the dark/inverted theme, the container fill is rgba(255,255,255,0.1). The selected segment has a #3389e0 fill with #ffffff text. Unselected segments have transparent fill with #a6b0be text. Unselected segments on hover use rgba(255,255,255,0.12) fill. When disabled, the selected segment fill is #64748b with #ffffff text. The label color is #ffffff.
