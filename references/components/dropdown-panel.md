Dropdown Panel (also referred to as "tree" on the design system site) is a hierarchical dropdown list for selecting items with optional search, grouping, and multiple selection modes. It accepts the following props: type ("Multi-Select" or "Single", defaulting to "Multi-Select"), size ("Medium" or "Small", defaulting to "Medium"), inverted (boolean, default false), nesting (boolean, default false), and showHeader (boolean, default true). It has four states: default, hover, selected, and disabled, and comes in two sizes: Medium and Small. Its anatomy consists of a panel with 8px corner radius, 8px horizontal padding, and a "0 8px 16px #1634621a" drop shadow; an optional header row (8px vertical padding, 8px gap) containing a select-all checkbox and a search input (12px horizontal padding, 6px gap, 8px corner radius, 1px border, with a 20x20 magnifying glass icon at 1.2 stroke width); and a list of rows with 2px gap and 8px bottom padding. Each row has 4px padding, 8px gap between children, 4px corner radius, and holds a checkbox (multi-select) or radio (single-select) icon plus a label that truncates with ellipsis; group parent rows show a 16x16 caret icon (1.5 stroke width) and use font weight 600 while regular row labels use weight 400, and group child rows are indented via left padding (56px at Medium, 48px at Small). At Medium size the panel is 244px wide with 32px rows, 20px selection icons, a 36px search input, and 14px/24px text; at Small it is 200px wide with 24px rows, 16px selection icons, a 28px search input, and 12px/16px text. The checkbox is a 20px square with 4px corner radius at Medium (16px with 3.2px radius at Small) and the radio a circle of outer radius 9 with a 4px inner dot at Medium (7 and 3 at Small), both at 1px stroke width. In the light theme the panel fill is #ffffff; default rows are transparent with #000000 text, hovered rows use #f7f8f9, selected rows use #e6f0fb background with #006bd8 text, and disabled rows use #a6b0be text at 0.5 opacity; checked checkbox/radio fill is #006bd8 (#64748b when disabled) with a #ffffff checkmark, unchecked stroke is #cbd1db (rgba(15,17,20,0.1) when disabled); caret and search icons are #64748b; the search input has an #e3e8ed border, #ffffff fill, and #64748b placeholder text. In the dark/inverted theme the panel fill is #475263; default rows use #ffffff text, hovered rows #2a313a, selected rows #003b77 background with #b0d1f3 text, and disabled rows #8390a2 text at 0.5 opacity; checked checkbox/radio fill is #3389e0 (#a6b0be when disabled), unchecked stroke #64748b (#5b6a7e when disabled); caret and search icons are #a6b0be; the search input has a #64748b border, transparent fill, and #a6b0be placeholder text. Guidelines: use multi-select for picking several items and single-select when only one choice is valid, enable nesting for hierarchical data, show the search header when the list has more than 7 items, and use Small in compact layouts like sidebars, popovers, or canvas areas; don't use it for flat lists with fewer than 5 items (use checkboxes or radios), don't nest more than 2 levels deep, don't hide the header when search matters for discoverability, and don't mix multi and single selection in the same instance.
